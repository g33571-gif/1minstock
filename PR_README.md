## ⚠️ 주의사항

### 1. 광고 차단(AdBlock) 영향
- 사용자가 AdBlock 켜면 광고 모두 안 보임
- 약 15~20% 사용자가 사용 → 정상

### 2. 애드핏 노출 조건
- localhost에서는 안 보임 (정상)
- 1minstock.com 배포 후에만 실제 광고 노출
- 광고단위 "심사 통과" 필수

### 3. 쿠팡은 즉시 동작
- 발급 즉시 실제 상품 표시
- 클릭 → 24시간 내 어떤 상품 사도 수수료 발생
